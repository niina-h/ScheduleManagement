# ScheduleManagement — 予定管理システム

チームメンバーが毎週の作業予定を登録・共有するための Web アプリケーションです。
ブラウザからアクセスでき、インストール不要で複数ユーザーが同時に利用できます。

---

## 機能概要

| 機能 | 説明 |
|------|------|
| ログイン | ユーザー名を選択してログイン。役職（マスタ／管理職／ユーザー）が自動反映される |
| 週間予定入力 | 月〜金の午前・午後スロット（各5枠）に作業名と作業時間を登録する |
| リアルタイム検証 | 基本勤務時間と合計が一致しない場合は赤文字で警告表示する |
| 先週コピー | ボタン1つで先週の予定をそのままコピーできる |
| 日次実績管理 | 当日の作業実績を記録し、繰越・週間予定への差し戻しに対応する |
| タスクマスター | よく使う作業名をユーザーごとに登録・管理し、予定入力時に選択できる |
| カテゴリ管理 | タスクの大分類・小分類を管理し、ドラッグ＆ドロップで表示順を変更できる |
| タスク管理 | プロジェクトタスクの登録・進捗管理。状態に応じた進捗率の自動計算に対応 |
| ブラビオ取込 | ブラビオExcelファイルからタスクを一括インポート。カテゴリ自動作成・既存タスク更新に対応 |
| ガントチャート | タスクをバー形式で表示。ドラッグで日程変更、状態・遅延・進捗のインライン編集が可能 |
| 全体ステータス | プロジェクト全体の進捗をグラフ・テーブルで俯瞰するダッシュボード |
| 進捗ダッシュボード | 担当者別の進捗状況を確認する画面 |
| 管理者ダッシュボード | 全ユーザーの週間予定登録状況を一覧で確認できる |
| ユーザー管理 | 管理者が担当者の追加・削除・基本勤務時間の変更を行える |
| 操作ログ | ログイン・データ保存・ユーザー操作などの操作履歴を記録・閲覧できる |
| メール日報 | 管理職が部下の日次実績をメール本文としてプレビュー・送信設定できる |
| Excel 出力 | 週間予定・ガントチャートを Excel ファイルとしてダウンロードできる |
| ヘルプ | アプリ内にヘルプ・手順書ページを内蔵。各機能の使い方を参照できる |

---

## 動作環境

- Python 3.9 以上
- Flask 3.0 以上
- openpyxl 3.1 以上

依存パッケージのインストール：

```bash
pip install -r requirements_web.txt
```

---

## 起動方法

```bash
python run_web.py
```

起動後、ブラウザで `http://localhost:5000` にアクセスしてください。

## 権限（3段階の役職体系）

| 役職 | できること |
|------|-----------|
| ユーザー | 自分の週間予定・日次実績の入力・編集、タスクマスター管理、自分の予定を Excel 出力 |
| 管理職 | 上記に加え、自部署メンバーの登録状況確認・編集、メール日報の作成・送信設定、タスク管理・ガントチャート・全体ステータス・進捗ダッシュボード |
| マスタ | 全ユーザーの参照・管理、部署マスタ管理、操作ログ閲覧、パスワード設定、全機能へのフルアクセス |

初期ユーザーは `data/users.json` に定義します。アプリ初回起動時に自動的にデータベースへ取り込まれます。
その後はブラウザの管理者ダッシュボードからユーザーを追加・削除できます。

## データファイル

| ファイル | 用途 | 備考 |
|----------|------|------|
| `data/users.json` | 初期ユーザーマスタ | 初回移行後は DB が正となる |
| `db/web_app.db` | アプリの全データ | 自動生成・自動更新 |
| `output/` | Excel・その他出力ファイル | 手動管理 |
