❓ ヘルプ・操作手順

🔑 ログイン手順
  1. ログイン画面でユーザー名を選択します。
  2. パスワード入力が必要なユーザーは4桁の数字を入力してください。
  3. 管理者アカウントは次回ログイン時にパスワードが自動入力されます（「パスワードは保存されています」と表示）。
  4. 「ログイン」ボタンを押すと週間予定画面へ移動します。
💡 パスワードを変更したい場合は管理者にお問い合わせください。
📅 週間予定の使い方
予定の入力
  1. 「週間予定」画面で該当週を表示します（◀前週・次週▶で週を移動）。
  2. 各曜日・午前/午後の枠にタスク名と時間を入力します。時間は 0.25時間（15分）単位 で入力してください。
  3. タスク名は「作業登録」で登録したタスクが補完候補に表示されます。
  4. 休暇がある場合は「休暇」プルダウンで種別を選択します（1日有休・AM半休など）。
  5. 入力後「保存」ボタンを押してください。
予定と実績の切り替え
  • 📋 予定ボタン：登録した週間予定を表示します。
  • 📊 実績ボタン：日次実績で登録した実績を表示します（当日以前の日のみ切り替わります）。
  • 日付の横に 実績 バッジがある日は実績が入力済みです。
その他の操作
  • 「先週の予定をコピー」：先週の予定をそのまま今週にコピーします。
  • 「📥 Excelダウンロード」：前週・今週・来週の3週分をExcelで出力します。📋予定／📊実績の表示状態に応じた内容で出力されます。
✏️ 日次実績の使い方
基本操作
  1. ナビバーの「本日実績」ボタン、または週間予定の日付リンクから実績入力画面を開きます。
  2. 左側「本日の予定」で当日の計画を確認しながら、右側「本日の実績」に実際の作業内容と時間を入力します。
  3. 時間は 0.25時間（15分）単位 で入力してください。
  4. 「保存」ボタンで実績を保存します。保存後、週間予定の「📊 実績」ボタンに反映されます。
⚠️ 実績は当日のみ編集可能です。過去の実績は管理者のみ修正できます。
📅 リスケ（予定していたが全く作業できなかった場合）
  1. 作業名欄をクリックすると「📅 リスケ」リンクが表示されます。
  2. クリックするとカレンダーが開くので、振り替え先の日付を選択します。
  3. 保存すると振り替え先の週間予定に「【MM/DD リスケ】作業名」が自動で追加されます。
  4. リスケした行の時間は自動的に0になります（未着手のため）。
  5. 作業名横のオレンジ色バッジ（MM/DD リスケ）をクリックするとリスケを解除できます。
⏩ 繰り越し（作業したが時間切れで終わらなかった場合）
  1. 作業名の右側にある薄い ⏩ ボタンをクリックして点灯（濃く）させます。
  2. 実績時間を入力して「保存」します。
  3. 保存後、画面下部に「⏩ 繰越業務（未完了）」カードが表示されます。
  4. 後日その作業が完了したら、カードの「解決」ボタンを押すと一覧から消えます。
💡 リスケと繰り越しの違い
📅 リスケ = 全く着手できなかった → 別日に振り替え（週間予定に自動追加）
⏩ 繰り越し = 着手したが未完了 → 繰越一覧で管理（週間予定への追加は手動）
実績クリアボタン
  • 「実績クリア」を押すと入力内容をすべて空にして保存します（振り返り・対策・上司コメントは保持されます）。
📥 Excel出力の使い方
  1. 週間予定画面の「📥 Excelダウンロード」ボタンをクリックします。
  2. 前週・今週・翌週の3シートが含まれるExcelファイルがダウンロードされます。
  3. 予定モード（📋 予定）で出力すると週間計画のみ記載されます。
  4. 実績モード（📊 実績）で出力すると実績入力済みの日は実績値で記載されます。
💡 出力前に「📋 予定」「📊 実績」ボタンで表示を切り替えると、その内容でExcelが出力されます。
⚙️ 管理者機能

管理者ロールのユーザーのみ利用できます。

ユーザー管理
  • メンバーの追加・削除・パスワード変更が行えます。
  • 標準作業時間（1日の目標時間）を個人ごとに設定できます。
  • 表示順の変更：ドラッグ＆ドロップまたは並び順設定でユーザーの表示順を変更できます。この順序はログイン画面・予定登録状況・Excel出力など全画面に反映されます。
  • 担当メンバー設定：管理職が閲覧・切り替え可能なメンバーを指定できます。
ユーザー切り替え
  • 管理職・マスタは週間予定・日次実績画面の右上「切り替え」ドロップダウンから他メンバーの画面を表示できます。
  • 管理職は自分自身＋担当メンバー、マスタは自分自身＋同部署全員が対象です。
メンバー全員エクスポート
  • 管理者ダッシュボードの「メンバー全員エクスポート」で同部署メンバーの今週予定をExcelで一括出力します。
  • 週間予定表テンプレート形式で、メンバーごとに10行（午前5枠＋午後5枠）で出力されます。
操作ログ
  • 管理画面の「📋 操作ログ」からメンバーのログイン履歴・操作履歴を確認できます。
  • アクション種別でフィルタして表示できます。
日次状況確認
  • 全メンバーの当日実績入力状況を一覧で確認できます。
