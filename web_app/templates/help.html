❓ ヘルプ・操作手順

🔑 ログイン手順
  1. ログイン画面でユーザー名を選択します。
  2. パスワード入力が必要なユーザーは4桁の数字を入力してください。
  3. 管理者アカウントは次回ログイン時にパスワードが自動入力されます（「パスワードは保存されています」と表示）。
  4. 「ログイン」ボタンを押すと週間予定画面へ移動します。
💡 パスワードを変更したい場合は管理者にお問い合わせください。
📅 週間予定の使い方
予定の入力
  1. 「週間予定」画面で該当週を表示します（◀前週・次週▶で週を移動）。
  2. 各曜日・午前/午後の枠にタスク名と時間を入力します。時間は 0.25時間（15分）単位 で入力してください。
  3. タスク名は「作業登録」で登録したタスクが補完候補に表示されます。
  4. 休暇がある場合は「休暇」プルダウンで種別を選択します（1日有休・AM半休など）。
  5. 入力後「保存」ボタンを押してください。
予定と実績の切り替え
  • 📋 予定ボタン：登録した週間予定を表示します。
  • 📊 実績ボタン：日次実績で登録した実績を表示します（当日以前の日のみ切り替わります）。
  • 日付の横に 実績 バッジがある日は実績が入力済みです。
その他の操作
  • 「先週の予定をコピー」：先週の予定をそのまま今週にコピーします。
  • 「📥 Excelダウンロード」：前週・今週・来週の3週分をExcelで出力します。📋予定／📊実績の表示状態に応じた内容で出力されます。
✏️ 日次実績の使い方
基本操作
  1. ナビバーの「本日実績」ボタン、または週間予定の日付リンクから実績入力画面を開きます。
  2. 左側「本日の予定」で当日の計画を確認しながら、右側「本日の実績」に実際の作業内容と時間を入力します。
  3. 時間は 0.25時間（15分）単位 で入力してください。
  4. 「保存」ボタンで実績を保存します。保存後、週間予定の「📊 実績」ボタンに反映されます。
💡 予定値の自動反映
実績が未登録の場合、予定の作業名・時間が自動的に実績欄に反映されます。そのまま保存すれば予定通りの実績として登録されます。
一度保存した後は、保存済みの実績値のみ表示されます（予定値は薄灰色のプレースホルダーとして参考表示）。
⚠️ 実績は当日のみ編集可能です。過去の実績は管理者のみ修正できます。
📅 リスケ（予定していたが全く作業できなかった場合）
  1. 作業名欄をクリックすると「📅 リスケ」リンクが表示されます。
  2. クリックするとカレンダーが開くので、振り替え先の日付を選択します。
  3. 保存すると振り替え先の週間予定に「【MM/DD リスケ】作業名」が自動で追加されます。
  4. リスケした行の時間は自動的に0になります（未着手のため）。
  5. 作業名横のオレンジ色バッジ（MM/DD リスケ）をクリックするとリスケを解除できます。
⏩ 繰り越し（作業したが時間切れで終わらなかった場合）
  1. 作業名の右側にある薄い ⏩ ボタンをクリックして点灯（濃く）させます。
  2. 実績時間を入力して「保存」します。
  3. 保存後、画面下部に「⏩ 繰越業務（未完了）」カードが表示されます。
  4. 後日その作業が完了したら、カードの「解決」ボタンを押すと一覧から消えます。
💡 リスケと繰り越しの違い
📅 リスケ = 全く着手できなかった → 別日に振り替え（週間予定に自動追加）
⏩ 繰り越し = 着手したが未完了 → 繰越一覧で管理（週間予定への追加は手動）
実績クリアボタン
  • 「実績クリア」を押すと実績を予定の値に戻して保存します。予定通りの状態から再編集できます（振り返り・対策・上司コメントは保持されます）。
📄 日報PDF / 📥 日報Excel / 📧 日報メール
  • 📄 日報 PDF：日報の印刷用ページが新しいタブで開きます。「🖨 印刷 / PDF保存」ボタンでブラウザの印刷ダイアログからPDF保存できます。
  • 📥 日報 Excel：日次業務報告をテンプレート形式のExcelでダウンロードします（1ページ印刷固定・翌日予定はカンマ区切り）。
  • 📧 日報メール：日報メールのプレビュー画面を開きます。宛先（TO/CC/BCC）・本文テンプレートを個別に設定・保存でき、「Outlookで開く」で件名・宛先・本文が設定された状態でOutlookが起動します。
  • 管理職・マスタは「📧 管理職日報メール」で管理職用・マスタ用のメールを作成できます。マスタ用は印刷専用の備考欄があり、印刷時のみメール下部に表示されます。
  • 管理職・マスタは「📋 チーム日報Excel」で担当メンバー全員分の日報を1つのExcelファイル（メンバー別シート）でダウンロードできます。
💡 PDF保存はブラウザの印刷機能を利用します。印刷ダイアログで「PDFに保存」を選択してください。
💡 金曜日のマスタ用メールには「管理業務のご報告」セクションが追加され、週間の大区分別実績が自動集計されます。
📝 作業登録の使い方
作業の登録
  1. ナビバーの「作業登録」からアクセスします。
  2. 作業名を入力し、大区分・中区分を選択します（任意）。
  3. デフォルト作業時間を設定すると、週間予定で自動入力されます（任意）。
  4. 「追加」ボタンで登録します。
登録済み作業一覧
  • 登録した作業がテーブルで一覧表示されます。
  • 大区分・中区分フィルター：テーブル上部のプルダウンで絞り込みできます。フィルター選択は削除・並び替え後も保持されます。
  • 列ソート：テーブルヘッダーをクリックすると昇順・降順で並び替えできます。
  • 並び順の変更：▲▼ボタンで表示順を変更できます。この順序が週間予定の入力候補の順番に反映されます。
  • 削除：不要な作業は削除ボタンで削除できます。
大区分・中区分の管理
  • 管理職・マスタ権限のユーザーは作業登録画面から大区分・中区分を追加・削除できます。
  • 大区分を削除すると、その配下の中区分もすべて削除されます。
📥 Excel出力の使い方
  1. 週間予定画面の「📥 Excelダウンロード」ボタンをクリックします。
  2. 前週・今週・翌週の3シートが含まれるExcelファイルがダウンロードされます。
  3. 予定モード（📋 予定）で出力すると週間計画のみ記載されます。
  4. 実績モード（📊 実績）で出力すると実績入力済みの日は実績値で記載されます。
💡 出力前に「📋 予定」「📊 実績」ボタンで表示を切り替えると、その内容でExcelが出力されます。
📧 メール日報の使い方
基本操作
  1. 日次実績画面の「📧 メール日報」ボタンからアクセスします。
  2. 当日の実績を元にメール本文が自動生成されます。
  3. 内容を確認し、「📨 Outlookで開く」ボタンを押します。
  4. 書式付き本文が自動でクリップボードにコピーされ、Outlookの新規メールが開きます（件名・宛先は自動設定、署名も保持）。
  5. メール本文欄で Ctrl+V で貼り付けてください（游ゴシック 11pt で貼り付けられます）。
💡 「📋 本文をコピー（テキスト）」ボタンでプレーンテキストとしてもコピーできます。
日報の内容
  • 業務内容：タスク管理画面で登録したプロジェクトタスクの進捗状況が表示されます。
  • 翌営業日の予定：翌営業日の週間予定が表示されます（定例作業は除外）。
  • 振り返り・対策：日次実績で入力した内容が反映されます。
メール設定
  • 管理画面からメール送信先（TO/CC/BCC）を設定できます。
  • BCC宛先は受信者に表示されないため、報告先の追加などに利用できます。
📋 タスク管理の使い方

管理職・マスタ権限のユーザーのみ利用できます。

概要

プロジェクト単位のタスクを登録し、期間と状態に基づいて進捗率を自動管理します。ここで登録したタスクがメール日報の業務内容に反映されます。

タスクの登録
  1. ナビバーの「タスク管理」からアクセスします。
  2. 「＋ タスク追加」ボタンで登録フォームを開きます。
  3. 大区分・中区分を選択し、タスク名（必須）・対応内容・開始日（必須）・終了日（必須）を入力します。
  4. 状態を選択し、必要に応じて遅延日数・進捗率を入力して「追加」ボタンを押します。
状態と進捗率の自動計算
状態進捗率
未着手0%（固定）
着手10%（固定）
順調経過日数 ÷ 全体日数 × 100（上限95%）
遅れ（経過日数 − 遅延日数）÷ 全体日数 × 100
完了100%（固定）
停止現在の進捗率で凍結
遅延日数
  • 「遅れ」状態のタスクに遅延日数を入力すると、その日数分を差し引いた進捗率が自動計算されます。
  • 例：全体30日・経過20日・遅延5日 → (20−5)÷30×100 = 50%
インライン編集
  • 一覧の各行で直接タスク名・対応内容・日付・状態・遅延日数・進捗率を編集し、「保存」ボタンで更新できます。
  • チェックボックスで選択した行を一括削除できます（選択行は赤くハイライトされます）。
関連画面
  • ガントチャート：タスクの日程をバー形式で視覚的に表示します（「ガントチャート」タブを参照）。
  • 全体ステータス：プロジェクト全体の進捗状況を俯瞰できます（「全体ステータス」タブを参照）。
  • 進捗ダッシュボード：担当者別の進捗状況を確認できます。
📊 ガントチャートの使い方

管理職・マスタ権限のユーザーのみ編集できます（一般ユーザーは閲覧のみ）。

基本操作
  • タスク管理画面の「ガントチャート」ボタン、またはナビバーからアクセスします。
  • 大区分でグループ化されたタスクがバーチャートで表示されます。
  • ◀前へ・次へ▶ ボタンで表示期間（4週間）をスクロールできます。
  • 「全開閉」ボタンでカテゴリの展開・折りたたみを一括切り替えできます。
  • カテゴリ行をクリックすると、そのカテゴリの展開・折りたたみを切り替えます。
左側列の表示
列内容編集
項目名タスク名を表示-
担当担当者名を表示-
状態タスクの状態（未着手・着手・順調・遅れ・完了・停止）クリックで変更
遅延進捗と日程から自動算出（遅れ: -N日 / 進み: +N日）自動計算
進捗進捗率（%）クリックで入力
バーのドラッグ操作（管理職・マスタのみ）
  • バー中央をドラッグ：タスクの日程を前後にスライドします（開始日・終了日が同時に変更）。
  • バー左端をドラッグ：開始日のみ変更します。
  • バー右端をドラッグ：終了日のみ変更します。
  • ドラッグ後に自動でサーバーに保存されます。
インライン編集（管理職・マスタのみ）
  • 状態・進捗の各セルをクリックすると、その場で編集できます。
  • 状態はプルダウンで選択、進捗率は数値を入力してEnterで確定します。
  • 遅延列は進捗率と日程から自動計算されます（遅れている場合は赤字で -N日、進んでいる場合は緑字で +N日）。
  • 変更は即座にサーバーに保存されます。
Excel出力
  • 「📥 Excel出力」ボタンで、現在表示中の期間のガントチャートをExcelファイルとしてダウンロードできます。
バーの色
  • 青：予定期間
  • 緑：進捗（実績）/ 完了タスク
  • 赤：遅延
  • グレー：停止中タスク
  • 紫破線：マイルストーン
  • 赤破線：今日の位置
📈 全体ステータスの使い方

管理職・マスタ権限のユーザーのみアクセスできます。

概要

プロジェクト全体のタスク進捗をグラフとテーブルで俯瞰できるダッシュボード画面です。

サマリーカード
  • 全タスク：登録されているタスクの総数
  • 完了：完了状態のタスク数
  • 遅れ：遅れ状態のタスク数
  • 全体進捗率：全タスクの平均進捗率
グラフ
  • ステータス別内訳：状態別のタスク数をドーナツチャートで表示
  • カテゴリ別 平均進捗率：大区分ごとの進捗率を横棒グラフで表示
テーブル
  • 担当者別タスク状況：担当者ごとの件数・完了数・遅れ数・平均進捗率を表示。担当者名をクリックすると進捗ダッシュボードへ移動します。
  • カテゴリ別詳細：大区分ごとの件数・完了数・遅れ数・平均進捗率を表示
  • 遅れ・注意タスク：遅れ状態のタスクまたは遅延日数が1日以上あるタスクの一覧
⚙️ 管理者機能

管理者ロールのユーザーのみ利用できます。

ユーザー管理
  • メンバーの追加・削除・パスワード変更が行えます。
  • 標準作業時間（1日の目標時間）を個人ごとに設定できます。
  • 表示順の変更：ドラッグ＆ドロップまたは並び順設定でユーザーの表示順を変更できます。この順序はログイン画面・予定登録状況・Excel出力など全画面に反映されます。
  • 担当メンバー設定：管理職が閲覧・切り替え可能なメンバーを指定できます。
ユーザー切り替え
  • 管理職・マスタは週間予定・日次実績画面の右上「切り替え」ドロップダウンから他メンバーの画面を表示できます。
  • 管理職は自分自身＋担当メンバー、マスタは自分自身＋同部署全員が対象です。
メンバー全員エクスポート
  • 管理者ダッシュボードの「メンバー全員エクスポート」で同部署メンバーの今週予定をExcelで一括出力します。
  • 週間予定表テンプレート形式で、メンバーごとに10行（午前5枠＋午後5枠）で出力されます。
操作ログ
  • 管理画面の「📋 操作ログ」からメンバーのログイン履歴・操作履歴を確認できます。
  • アクション種別でフィルタして表示できます。
日次状況確認
  • 全メンバーの当日実績入力状況を一覧で確認できます。
